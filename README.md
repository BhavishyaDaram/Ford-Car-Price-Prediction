# 🚗 Ford Car Price Prediction

## Overview
This project predicts the selling price of used Ford cars using Machine Learning.

## Dataset
- Ford Used Car Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Workflow
- Data Cleaning
- Exploratory Data Analysis
- Feature Encoding
- Train-Test Split
- Linear Regression Model
- Model Evaluation

## Evaluation Metrics
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- R² Score

## Results
The model successfully predicts used car prices based on various vehicle features.
